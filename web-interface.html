How to Use

This tool converts Hindi and Urdu text into a standardized Latin script that preserves all the phonetic features of the original text.

Key Features:

  • Uses Turkish letters (ç, ş, etc.) for certain sounds
  • Uses IPA symbols for retroflexive sounds
  • Uses diacritics for aspirated, nasalized, and long sounds

Character Guide:

Original Latin Description
ج / ज c Turkish c for j sound
چ / च ç Turkish ç for ch sound
ش / श ş Turkish ş for sh sound
ٹ / ट ʈ IPA for retroflex t

About This Project

The Hindi-Urdu Latinization project aims to create a standardized system for representing Hindi and Urdu in the Latin script, preserving all phonetic features of these languages.

Features:

  • Consistent representation of sounds across both languages
  • Preservation of phonetic distinctions using diacritics
  • Open-source implementation
